# 共享知识库

跨 Agent 共享的知识沉淀，包含通用的最佳实践、架构决策和常见问题解决方案。

## 知识分类

知识库按主题组织，每个主题下包含相关的知识条目。与工作区类似，知识库条目也可以通过**软链接**关联到各 Agent 项目中的原始文档。

```bash
# 链接 Agent 的共享知识文件
ln -s /path/to/agent/knowledge/patterns.md content/docs/knowledge-base/patterns.md
```
